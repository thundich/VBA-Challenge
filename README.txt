# VBA Module Challenge 2

  Tamara Hundich

  09/27/2023

  Michigan State University edX Data Analytics Bootcamp 

  Visual Basic Module Challenge 2 of the Multiple Year Stock Data that demonstrates the usage of for looping and if/then/else logic control

## Requirements
  - creating scripts that loops through all the stocks for one year and outputs the following information:

		- The ticker symbol

		- Yearly change from the opening price at the beginning of a given year to the closing price at the end of that year.

		- The percentage change from the opening price at the beginning of a given year to the closing price at the end of that year.

		- The total stock volume of the stock. 
		
	- create a loop to retrieve minimum and maximum data 
	
	- program must be able to run on multiple worksheets 


## Installation and Configuration 
	- No installation is required. 
	
	- Files included with repository include the VBA file code, README, and resulting images of yearly findings.
